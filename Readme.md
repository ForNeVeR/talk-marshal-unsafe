talk-marshal-unsafe
===================

This is a talk (in Russian) about .NET marshaling and unsafe code.

How to install
--------------

```console
$ npm ci
$ # open ./index.html to view in browser
```

To read the talk as an article, open [talk.md][].

All the code samples are available in `src` directory.
